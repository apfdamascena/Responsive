# Responsive

Layout :

-Fixo 
`px` - Pixels

-Fluido
`%` - Porcentagem
`auto` - Automatico
`vh` - viewport height
`vw` - viewport width

Texto :

-fixos
`1px` = 0.75pt

-fluidos
`em` - multiplicado pelo pai
`rem` - multiplicado pelo root

- primeira estrategia:
    colocar width : 100%
    e caso precise, definir o max-width. por exemplo: max-width:980px;

- segunda estrategia:
    definir body com font-size: 62.5% e trabalhar o resto do texto com 1.0rem para 10px;
